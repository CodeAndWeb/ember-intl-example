## Installation of the example project

* `git clone https://github.com/CodeAndWeb/ember-intl-example.git` this repository
* `cd ember-intl-example`
* `npm install`

## Running / Development

* `ember serve`
* Visit your app at [http://localhost:4200](http://localhost:4200).
